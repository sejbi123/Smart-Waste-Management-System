# 👤 User Stories

## 🟢 Waste Management Manager
- As a Waste Management Manager, I want to monitor bin fill levels, truck locations, and schedules in real time, so that I can make informed decisions and optimize operations.
- As a Waste Management Manager, I want to approve and adjust collection schedules, so that resources are used efficiently.
- As a Waste Management Manager, I want to view analytics and reports, so that I can evaluate performance and improve service quality.

---

## 🚛 Truck Driver
- As a Truck Driver, I want to receive optimized routes on a mobile device, so that I can complete collections efficiently.
- As a Truck Driver, I want to mark bins as collected and report issues (e.g., damaged bins), so that the system remains accurate and up to date.
- As a Truck Driver, I want clear step-by-step navigation, so that I can avoid delays and confusion.

---

## 🧰 Field Operator (Bin Technician)
- As a Field Operator, I want to receive maintenance tasks with bin locations, so that I can fix issues quickly.
- As a Field Operator, I want to update bin status (damaged, repaired, replaced), so that the system reflects real conditions.
- As a Field Operator, I want to upload photos of issues, so that problems are clearly documented.

---

## 🏭 Waste Disposal Facility Manager
- As a Facility Manager, I want to see incoming trucks and their schedules, so that I can manage facility capacity.
- As a Facility Manager, I want to record waste quantities and types, so that disposal and recycling are tracked accurately.
- As a Facility Manager, I want to identify contaminated waste, so that recycling quality is maintained.

---

## 👥 Citizen of Tirana
- As a Citizen, I want to report full bins or illegal dumping, so that issues are resolved quickly.
- As a Citizen, I want to track the status of my report, so that I know my request is being handled.
- As a Citizen, I want a simple and accessible reporting interface, so that I can easily submit issues.

---

## 🏛️ ECO Tirana Management
- As ECO Tirana Management, I want to view performance reports and KPIs, so that I can evaluate system effectiveness.
- As ECO Tirana Management, I want to monitor operational costs, so that I can reduce expenses.
- As ECO Tirana Management, I want reliable and accurate data, so that I can support strategic decision-making.

---

## 🏢 Municipality of Tirana
- As the Municipality, I want to receive accurate waste management reports, so that I can ensure compliance with regulations.
- As the Municipality, I want insights into recycling rates and environmental impact, so that I can support sustainability goals.
